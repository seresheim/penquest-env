Welcome to PenQuest-Env's documentation!
========================================

PenQuest-Env is a reinforcement learning environment for the digital board game
`PenQuest <https://www.pen.quest/>`_. The environment uses a gymnasium v0.26 API,
for more information on this see `the gymnasium website 
<https://gymnasium.farama.org/>`_.

.. note::

   In order to use PenQuest-Env you need an API key. Please contact the
   developers in order to get one.
